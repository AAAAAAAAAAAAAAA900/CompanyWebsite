Photo King: Image Sharing Game

'Photo King: Image Sharing Game' is a social media app built in react native and spring boot. It allows users to create accounts, add friends, invite friends to groups, share photos, and rank photos to crown a photo king. Groups reset every week and assign the top 3 placing image posters points tracked on the group members bar. Additional features include profile customization, image comment sections, AI image filtering using OpenAI's moderation API, blocking users, and reporting photos/comments. Check us out on the app store here: Photo King: Image Sharing Game

To Remove Account and Associated Data

To manually delete all your data: log into the app, tap the profile icon on the top right of the header bar, and select "delete account". To request your data to be deleted: please email condoncorp5@gmail.com using the subject line "delete account" either from the accounts associated email address or including the account credentials in the body of the email. Using either method, we will retain none of your data after processing the successful account deletion.
